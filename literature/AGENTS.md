# literature/

Paper digests, reading lists, structured notes.

- **Agent:** `.cursor/agents/literature.md`
- **Prefer:** Cloud (async digests → markdown PR)
- Cite sources; one topic per run

## Math in markdown

Use GitHub-friendly delimiters so equations render on GitHub and in most previews:

- Inline: `$...$` (not `\(...\)`)
- Display: `$$...$$` on their own lines (not `\[...\]`)
